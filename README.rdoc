== wyrd

wyrd is a web application that allows users to create shelves of books based on criteria of their choosing. wyrd users can search for books using the app's implementation of the GoogleBooks API. This allows a user to add nearly any book to his or her shelves!

wyrd was developed as a project at General Assembly in Atlanta, GA over a week-long sprint

Technologies used:

* Ruby

* Rails

* Bootstrap

* PostgreSQL

* GoogleBooks API

* RSpec
